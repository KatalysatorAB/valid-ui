Country is a Select component that contains valid compatible countries.

## When to use?
When combined `<SelectWithCountry />` is not enough for you, or you want to have a better control over form items. Use together with `<Select />` component.

## Props

| Property name      | Description                                                                  | Type                                                           | Required |
| :---               | :----                                                                        | :----                                                          | :---- |
| countries          | Countries array that specifies available countries | `CountryCode[]`         | no |
| countryMode        | Mode of which Select will be displayed                                       | `'text', 'icons', 'icons-text', 'text-icons'`                                                       | yes |
| onSelect           | Called when selecting option                                                 | `(country: CountryCode) => void`                   | yes |
| countryPlaceholder | Placeholder text that will be displayed in the component                     | `string`                                                       | no |
